Kata Score es una aplicación web de calificación para torneos de Kata.
El kata es una de las modalidades principales del karate.
Se trata de una secuencia preestablecida de movimientos que simulan un enfrentamiento
Cada kata tiene un nombre, una serie específica de técnicas y movimientos que deben ser ejecutados con precisión.

Esta aplicación web con un entorno muy disfrutable para los jueces y el publico permite:
Realizar un sorteo, generando los enfrentamientos de la competencia por categorias.
Permite a los jueces calificar y descalificar a competidores.
